# CRC_iOS
* Sensor data logger for iOS
* Collects data for 660 seconds and saves in .csv file
* Data is saved once in 5 seconds
* Converted units to match with the Android app

## List of sensor data collected by the app
### Common Data 
- Elapsed Time 
- World Time `Year`
